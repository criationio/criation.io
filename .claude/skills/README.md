# Skills do Claude Code — Criation.io

Esta pasta contem skills customizadas para uso nas sessoes de desenvolvimento do Criation.io com Claude Code.

## Skills ativas

| Skill | Localizacao | Descricao |
|---|---|---|
| `frontend-design` | skill global Anthropic | Design system, tokens, componentes |
| `prompt-engineering` | skill global Anthropic | Prompts para Claude API |

## Como adicionar uma skill

1. Crie uma pasta em `.claude/skills/nome-da-skill/`
2. Crie `SKILL.md` com instrucao de uso
3. Documente aqui neste README
