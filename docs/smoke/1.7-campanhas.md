# Smoke E2E — Sessão 1.7 (/campanhas listagem + detalhe + comparativo)

> Executar manualmente após `pnpm dev`. Marcar ✅ ou descrever defeito encontrado.

## Pré-requisitos

- [ ] Login em workspace com pelo menos 1 conexão Meta Ads ativa
- [ ] Sync rodou ao menos 1× (campanhas existem em `campaigns`)
- [ ] Ideal: workspace tem gateway_events com `matched_campaign_id` populado (para ver receita/ROAS)

## Área 1 — Listagem /campanhas

- [ ] Página renderiza header "Campanhas" + descrição
- [ ] Botão "Sincronizar agora" aparece quando há connection ativa
- [ ] Filtros bar mostra: Período (dropdown), Status (dropdown), Plataforma (dropdown), Busca (input), botão Limpar (aparece só com filtros aplicados)
- [ ] Tabela tem 10 colunas: Nome, Plataforma, Status, Gasto, Impressões, CTR, Conv., Receita, ROAS, Última sync
- [ ] Nome clicável navega pra `/campanhas/[id]`
- [ ] ROAS verde quando ≥ 2×, vermelho quando < 1×, laranja entre
- [ ] Paginação aparece quando total > 25, mostrando "X–Y de Z" + setas Prev/Next
- [ ] Paginação desabilita Prev na página 1 e Next na última página

## Área 2 — Filtros aplicam corretamente

- [ ] Mudar período `last_30d → last_7d` atualiza URL (`?period=last_7d`) e dados na tabela mudam
- [ ] Selecionar Status "Ativa" filtra apenas campaigns ACTIVE
- [ ] Selecionar Plataforma "Meta" filtra apenas Meta
- [ ] Digitar texto na busca + Enter (ou blur) filtra por nome (ILIKE %texto%)
- [ ] Trocar página preserva filtros na URL
- [ ] Mudar filtro reseta para página 1
- [ ] Clicar Limpar restaura URL para `/campanhas` (sem params)

## Área 3 — Detalhe /campanhas/[id]

- [ ] Link "Voltar para campanhas" funciona
- [ ] Header mostra nome + status pill + provider + objective + budget
- [ ] Filtro de período no canto direito (dropdown 10 opções)
- [ ] 8 KPI cards renderizam: Gasto, Receita, Cliques, CTR, Conversões, ROAS, Impressões, CPA
- [ ] Cada KPI card mostra valor + delta % vs período anterior + sparkline
- [ ] CPA tem polaridade invertida (queda = verde, alta = vermelho)
- [ ] Cards com value 0 mostram "—" + "Sem dados no período"
- [ ] 404 quando campaignId não existe ou não pertence ao workspace

## Área 4 — Chart timeseries

- [ ] Gráfico Recharts renderiza com:
  - Area verde: Receita
  - Line laranja: Gasto
  - Line accent tracejada: Cliques (eixo direito)
  - Line preta: Conversões (eixo direito)
- [ ] Tooltip mostra todas 4 séries com formato BRL ou número
- [ ] Empty state quando todos os dias têm zero

## Área 5 — Tabela ad_sets aninhada

- [ ] Tabela mostra todos ad_sets da campaign
- [ ] Cada linha tem chevron right/down clicável
- [ ] Click expande/colapsa linha, mostrando ads abaixo com bullet "↳"
- [ ] Ads expandidos têm estilo distinto (bg-subtle, fonte menor)
- [ ] Colunas alinhadas: Nome, Status, Gasto, Impressões, Cliques, CTR, Conv., Receita
- [ ] Empty state "Nenhum conjunto de anúncios sincronizado"

## Área 6 — Galeria de criativos

- [ ] Grid responsivo: 2 cols mobile, 5 cols desktop
- [ ] Card mostra: thumbnail (ou placeholder ImageIcon), título/ad_name, gasto, CTR, impressões, cliques
- [ ] Badge "Vídeo" aparece quando creative tem video_url
- [ ] Thumbnails carregam direto da URL do Meta (sem next/image — domínio externo)
- [ ] Empty state "Nenhum criativo sincronizado pra essa campanha"

## Área 7 — Comparativo A×B

- [ ] Página /campanhas/comparativo renderiza header + 2 dropdowns + dropdown período
- [ ] Dropdown A não mostra a campanha já selecionada em B (e vice-versa)
- [ ] Botão swap (ArrowLeftRight) troca A e B
- [ ] Empty state "Selecione duas campanhas acima" quando A ou B vazio
- [ ] Quando ambos selecionados, renderiza tabela com 10 métricas: Gasto, Receita, Impressões, Cliques, CTR, Conversões, CPA, CPC, CPM, ROAS
- [ ] Coluna do vencedor por métrica aparece em verde negrito
- [ ] CPA/CPC/CPM: menor vence (lowerIsBetter)
- [ ] Receita/ROAS: maior vence
- [ ] Coluna Δ B vs A mostra % com seta up/down/flat e cor verde/vermelho/cinza

## Área 8 — Performance + UX

- [ ] Build de produção (`pnpm build`) compila sem erros
- [ ] Páginas SSR não disparam hydration mismatch (console limpo)
- [ ] Tabela com 25 linhas renderiza < 200ms (não há jank perceptível)
- [ ] Loading states gracioso quando navegando entre páginas
- [ ] Dark mode + light mode renderizam corretamente todos componentes

## Edge cases conhecidos

- [ ] Campanha sem ad_insights no período: KPIs zeram, "—" no lugar de R$ 0,00
- [ ] Campanha sem ads (raro): tabela vazia + empty state
- [ ] gateway_events vazio: Receita/ROAS/Conversões = 0, "—" displayed
- [ ] Filtro período retornando 0 campanhas: empty state contextualizado
